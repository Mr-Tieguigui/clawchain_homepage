Product Intro

Built for the moment when an agent has enough privilege to break real things.

ClawChain is not a passive logger. It sits on the execution path of live coding-agent work, keeps recovery material near the operator, exposes the dangerous path in a readable console, and turns intervention, restore, proof export, and verification into one continuous flow.

Live Control

Discover and route active sessions instead of forcing a separate workflow.

Recovery First

Snapshot-backed recovery is preserved before deletes, moves, truncates, and integrity mutations land.

Proof Path

Receipts, impact sets, and anchor verification remain readable enough for operators to trust.

Clone and install once, start the project, then choose the host-specific setup command that matches your machine.

Control Sequence

From detection to verified recovery.

The control loop is designed so the operator can move through the risky path without losing context or evidence.

01

Discover

Locate live high-privilege agent sessions already active on the host.

02

Route

Switch into a managed launcher path with resume and handoff commands.

03

Capture

Detect destructive actions early enough to preserve snapshot-backed recovery material.

04

Restore

Recover the affected target from the vault and validate the restored state.

05

Verify

Export proof and check anchor fields against the configured chain backend.
